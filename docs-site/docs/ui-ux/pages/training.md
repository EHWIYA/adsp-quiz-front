# Training 페이지

## 개요
학습 모드 페이지. 문제 풀이 및 즉시 피드백 제공.

## 레이아웃 구조

```
┌─────────────────────────┐
│   Header                │
│   - 제목: 학습 모드     │
└─────────────────────────┘
┌─────────────────────────┐
│   필터 섹션            │
│   - 과목 Dropdown       │
│   - 주요항목 Dropdown   │
│   - 세부항목 Dropdown   │
└─────────────────────────┘
┌─────────────────────────┐
│   QuestionDisplay       │
│   - 문제 카드          │
│   - 선택지             │
│   - 해설 (정답 표시 시)│
│   - 수정 요청 버튼     │
└─────────────────────────┘
┌─────────────────────────┐
│   Footer (조건부)      │
│   - 다음 문제 버튼     │
│   (정답 표시 후 표시)  │
└─────────────────────────┘
```

## 사용자 플로우
1. 과목/주요항목/세부항목 선택 (선택사항)
2. 문제 표시
3. 답안 선택
4. 자동으로 정답 표시 및 해설 노출
5. 오답인 경우 오답노트에 자동 저장
6. "다음 문제" 버튼 클릭
7. 다음 문제로 이동

## 특징
- 즉시 피드백 제공
- 해설 자동 표시
- 3단계 분류 필터 (과목 > 주요항목 > 세부항목)
- 핵심 정보 기반 문제 추천
- 문제 수정 요청 기능
- 오답 자동 저장
