# Training 페이지

## 개요
학습 모드 페이지. 문제 풀이 및 즉시 피드백 제공.

## 레이아웃 구조

```
┌─────────────────────────┐
│   Header                │
│   - 제목: 학습 모드     │
│   - 진행률: 문제 N/10   │
└─────────────────────────┘
┌─────────────────────────┐
│   QuestionDisplay       │
│   - 문제 카드          │
│   - 선택지             │
│   - 해설 (정답 표시 시)│
└─────────────────────────┘
┌─────────────────────────┐
│   Footer (조건부)      │
│   - 다음 문제 버튼     │
│   (정답 표시 후 표시)  │
└─────────────────────────┘
```

## 사용자 플로우
1. 문제 표시
2. 답안 선택
3. 자동으로 정답 표시 및 해설 노출
4. "다음 문제" 버튼 클릭
5. 다음 문제로 이동

## 특징
- 즉시 피드백 제공
- 해설 자동 표시
- 진행률 표시
