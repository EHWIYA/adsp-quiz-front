# Exam 페이지

## 개요
모의고사 페이지. 시간 제한이 있는 시험 환경 제공.

## 레이아웃 구조

```
┌─────────────────────────┐
│   Header                │
│   - 제목: 모의고사      │
│   - Timer (일시정지/재개)│
└─────────────────────────┘
┌─────────────────────────┐
│   Progress Bar          │
│   - 문제 번호: N/50     │
│   - 남은 시간           │
│   - 진행률 바           │
└─────────────────────────┘
┌─────────────────────────┐
│   QuestionDisplay       │
│   - 문제 카드          │
│   - 선택지             │
└─────────────────────────┘
┌─────────────────────────┐
│   Footer                │
│   - 시험 제출 버튼     │
└─────────────────────────┘
```

## 주요 기능
- 타이머 (90분 제한)
- 일시정지/재개 기능
- 진행률 표시
- 시험 제출

## 특징
- 정답 표시 없음 (학습 모드와 차이)
- 시간 제한 적용
- 제출 전 확인 모달 (추후 구현)
