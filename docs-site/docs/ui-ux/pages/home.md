# Home 페이지

## 개요
서비스 메인 대시보드 페이지. 통계 정보 및 주요 액션 제공.

## 레이아웃 구조

```
┌─────────────────────────┐
│   Hero Section         │
│   - 제목               │
│   - 부제목             │
└─────────────────────────┘
┌─────────────────────────┐
│   통계 카드 (3개)      │
│   - 풀이한 문제        │
│   - 평균 정답률        │
│   - 완료한 모의고사    │
└─────────────────────────┘
┌─────────────────────────┐
│   액션 버튼            │
│   - 학습 모드 시작     │
│   - 모의고사 시작      │
└─────────────────────────┘
┌─────────────────────────┐
│   최근 활동            │
│   - 활동 내역 리스트   │
└─────────────────────────┘
```

## 주요 기능
- 통계 대시보드 (현재 더미 데이터)
- 학습 모드/모의고사 바로가기
- 최근 활동 내역 표시

## 반응형
- 모바일: 세로 스택 레이아웃
- 태블릿/데스크톱: 가로 배치
