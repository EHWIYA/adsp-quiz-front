# Admin 페이지

## 개요
관리자 페이지. Tab 컴포넌트를 사용하여 여러 관리 기능을 제공.

## 기능
- 대시보드: 통계 및 개요
- 핵심 정보 등록: CoreContentRegistration
- 문제 관리: QuizManagement

## 라우팅
- 경로: `/admin`

## 주요 컴포넌트
- `Tab`: 탭 인터페이스
- `Dashboard`: 대시보드 컴포넌트
- `CoreContentRegistration`: 핵심 정보 등록 컴포넌트
- `QuizManagement`: 문제 관리 컴포넌트

## 탭 구조
1. **대시보드**: 전체 통계 및 개요
2. **핵심 정보 등록**: 핵심 정보 등록 및 관리
3. **자동 분류 검수**: 보류 항목 승인/반려
4. **자동 분류 설정**: 임계값·전략·가중치 설정
5. **문제 관리**: 문제 CRUD 및 수정 요청 처리

## 핵심 정보 등록
- 등록 방식 선택: 카테고리 지정 / 자동 분류
- 자동 분류: 카테고리 미지정, 텍스트/YouTube URL만 입력
- 카테고리 지정: 과목/주요항목/세부항목 선택 후 등록
- 등록 후 분류 결과 확인 및 저장/보류 선택

## 자동 분류 관리
- 임계값 미달 건은 검수 대기 목록으로 이동
- 검수 승인/반려 및 오버라이드 기록
- 운영 설정 조정(임계값, 전략, 가중치, 후보 수)
- 검수 사유(reason) 입력 지원
- 후보 카테고리 트리 미리보기 제공
- 승인 사유 필수(최소 3자), 보류 사유 필수(최소 10자), 최대 200자 제한
- 승인/보류 사유 분리 입력
- 최소 글자수는 검수 UI 설정에서 조정 가능
- 하이라이트 색상/시간 프리셋 설정 가능
- 사유 입력 템플릿 버튼 제공
- 템플릿 목록은 검수 UI 설정에서 편집 가능
- 템플릿 즐겨찾기는 검수 탭에서 설정
- 하이라이트 커스텀(색상/시간 직접 입력) 지원

## 상태 관리
- 탭 활성 상태 관리
- 각 탭별 독립적인 상태 관리
