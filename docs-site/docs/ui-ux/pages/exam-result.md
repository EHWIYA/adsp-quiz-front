# ExamResult 페이지

## 개요
모의고사 결과를 표시하는 페이지. 시험 세션 ID를 통해 결과를 조회하고 상세 정보를 표시.

## 기능
- 시험 결과 통계 표시 (ResultAnalysis 컴포넌트 사용)
- 문제별 정답 확인
- 정답/오답/미답 구분 표시
- 문제별 해설 표시
- 다시 시험 보기 / 마이페이지 이동 버튼

## 라우팅
- 경로: `/exam/result/:examSessionId`
- 파라미터: `examSessionId` (시험 세션 ID)

## 주요 컴포넌트
- `ResultAnalysis`: 결과 통계 표시
- `Button`: 네비게이션 버튼

## 상태 관리
- `useExamResult`: 시험 결과 조회 API
- 로딩/에러 상태 처리

## 사용 흐름
1. 모의고사 완료 후 결과 페이지로 이동
2. 시험 세션 ID로 결과 조회
3. 통계 및 문제별 상세 정보 표시
4. 다시 시험 보기 또는 마이페이지로 이동
