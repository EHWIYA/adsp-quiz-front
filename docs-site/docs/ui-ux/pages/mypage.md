# Mypage 페이지

## 개요
사용자 정보 및 학습 통계를 표시하는 마이페이지.

## 레이아웃 구조

```
┌─────────────────────────┐
│   Header                │
│   - 제목: 마이페이지    │
└─────────────────────────┘
┌─────────────────────────┐
│   Profile Card          │
│   - 아바타             │
│   - 사용자명           │
│   - 이메일             │
└─────────────────────────┘
┌─────────────────────────┐
│   통계 카드 (4개)      │
│   - 총 풀이 문제       │
│   - 평균 정답률        │
│   - 완료한 모의고사    │
│   - 연속 학습일        │
└─────────────────────────┘
┌─────────────────────────┐
│   최근 모의고사 결과    │
│   - 결과 리스트        │
│   - ResultAnalysis      │
└─────────────────────────┘
┌─────────────────────────┐
│   설정                  │
│   - 알림 설정          │
│   - 학습 목표 설정     │
│   - 계정 설정          │
└─────────────────────────┘
```

## 주요 기능
- 사용자 프로필 표시
- 학습 통계 대시보드
- 최근 시험 결과 목록
- 설정 메뉴
